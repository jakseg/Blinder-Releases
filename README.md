# Blinder

A minimalist Pomodoro timer that lives in your macOS menu bar.

## Features

- Pomodoro timer with configurable work/break durations
- Short and long break intervals
- Task management with progress tracking
- Daily goal tracking
- Import tasks from Notion, Reminders, and Notes
- Notification support for work/break transitions

## Installation

1. Download the latest `.dmg` from [Releases](https://github.com/jakseg/Blinder-Releases/releases/latest)
2. Open the DMG and drag **Blinder** to your Applications folder
3. Open Terminal and run:
   ```bash
   xattr -cr /Applications/Blinder.app
   ```
   This is required because the app is not notarized by Apple.
4. Open **Blinder** from your Applications folder

## System Requirements

- macOS 26.2 or later
